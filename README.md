### How to use it?
1. Open terminal and go to root
```bash
cd ~
```
2. Clone this repo:
```bash
git clone https://github.com/RC-Sho0/Sho-dot-file.git
```
3. Move all file to root
```bash
mv ~/Sho-dot-file/.* ~/
mv .config .oh-my-zsh .zsh ~/
```

4. Config nvim
```bash
nvim +PackerSync
```
